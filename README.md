Food Donation Analytics & Prediction System

Developed a data-driven system to analyze food donation listings and assist NGOs in optimizing pickup scheduling and reducing food waste. The solution predicts food freshness and the likelihood of successful NGO collection, enabling smarter decision-making for time-sensitive donations.

An interactive Streamlit dashboard provides real-time insights and predictions, making the system intuitive and practical for NGO operations.

Objectives
Analyze historical food donation data
Predict food freshness using key features
Estimate the likelihood of NGO pickup/collection
Prioritize high-value and time-critical donations

Key Features
Interactive analytics dashboard using Streamlit
Real-time prediction of food freshness
Pickup/collection likelihood prediction
Data-driven insights for optimized NGO scheduling
Machine learning–based prediction system

Tech Stack
Python
Pandas, NumPy
Scikit-learn
Streamlit
Matplotlib, Seaborn

Impact
Enabled faster and more informed decision-making for NGOs
Improved prioritization of urgent food pickups
Reduced food wastage through timely intervention
Enhanced efficiency of food distribution systems
